# 空·Rhythm

> 随节拍而舞，逐音入虚空。
> *Move to the rhythm. Chase the void.*

**空·Rhythm** is a browser-based rhythm game built with pure HTML/CSS/JS. Tap notes as they fall to the beat — build combos, set high scores, lose yourself in the rhythm.

## How to Play

- **Tap/Click** the colorful notes as they reach the hit zone
- Notes fall in **3 lanes** (left, center, right) — track them all!
- **PERFECT** (within 4%) -> +3pts x multiplier, combo up
- **GOOD** (within 9%) -> +1pt x multiplier, combo up
- **MISS** -> combo resets to 0
- **Combo multiplier**: x1 (<10), x2 (10+), x3 (25+), x4 (50+)
- **Esc** to end game anytime
- High score saved in browser

## Features

- 3-lane note system with trail effects
- Procedural audio (kick, snare, hi-hat via Web Audio API)
- Combo multiplier with visual tiers (hot / fire)
- Beat flash effect on perfect/good hits
- BPM accelerates 100 -> 180 as you play
- Starfield particle background (150 stars)
- Combo milestone sound effects
- Responsive, works on mobile + desktop
